<h1>🎬 NOTFlix</h1>

Uma aplicação web inspirada em plataformas de streaming como a Netflix, desenvolvida durante a Imersão Front-End com IA da Alura.

O projeto simula uma experiência completa de usuário, desde a seleção de perfis até a navegação em um catálogo dinâmico de conteúdos, com foco em interatividade, organização de dados e construção de interfaces modernas.

🔗 Acesse o projeto:
👉 https://thesynnc.github.io/imersao-frontend-ia/

<h1>🚀 Funcionalidades</h1>  
Seleção de Perfis: Tela inicial interativa para escolha de usuário
Gerenciamento de Perfis: Navegação para edição/controle de perfis
Catálogo Dinâmico: Conteúdos organizados em sliders horizontais (estilo Netflix)
Navbar Completa: Navegação entre seções como início, séries, filmes e lista pessoal
Interface Interativa: Elementos visuais com destaque, ícones e feedback ao usuário
Sistema de Notificações: Simulação de alertas no topo da aplicação
Renderização com JavaScript: Conteúdo carregado dinamicamente sem reload
Responsividade: Layout adaptado para diferentes dispositivos

<h1>🛠 Tecnologias</h1>
HTML5: Estrutura das páginas
CSS3: Estilização moderna e responsiva (Flexbox/Grid)
JavaScript (ES6+): Manipulação de DOM e lógica da aplicação
Módulos JavaScript: Organização do código em arquivos separados
Font Awesome: Ícones da interface
Google Fonts (Roboto): Tipografia
